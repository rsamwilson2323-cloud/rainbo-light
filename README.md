# 🌈 Rainbo Light

A mesmerizing animated gradient background built with **HTML5 Canvas** and **JavaScript**. The project creates a continuously flowing rainbow light effect using dynamic color transitions and smooth gradient animations across the entire screen.

## ✨ Features

* 🌈 Smooth animated rainbow gradients
* 🎨 Dynamic color transitions using HSL
* ⚡ Real-time canvas rendering
* 📱 Fully responsive fullscreen design
* 🖥️ Works on desktop and mobile browsers
* 🚀 Lightweight and dependency-free
* 🎭 Endless looping animation
